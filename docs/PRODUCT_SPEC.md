# Healthcare Data Workflow Demo

## Product Goal
Build a lightweight proof-of-concept web app that demonstrates a mock healthcare data workflow.

This is not a production healthcare system.

The app demonstrates:
- patient consent workflow
- simulated healthcare data retrieval
- audit logging
- encounter context comparison
- data completeness concepts

## Tech Scope
- React or Next.js
- Mock JSON data only
- No real healthcare API
- No authentication
- No payments
- No claims processing

## Core Screens
1. Connect
2. Consent
3. Data Display
4. Audit Log
5. Encounter Context Comparison

## Core Entities
- Patient
- Consent
- HealthcareData
- Encounter
- AuditLog

## Success Criteria
A user can:
1. Start connection flow
2. Give mock patient consent
3. View simulated patient data
4. See audit log events
5. Compare two encounter contexts
